#ReactNative 之活动指示器(菊花)自定义、颜色等相关属性了解
<h3>ReactNative---------->活动指示器属性如下</h3>
<ul type="disc">
 <li>animating:是否要显示指示器，类型为bool 默认为true。</li>
 <li>hidesWhenStopped:在没有动画的时候，是否要隐藏指示器（默认为true） </li>
 <li>size:指示器的大小。small的高度为20，large为36</li>
 <li></li>
</ul>
